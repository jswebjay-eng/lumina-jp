# Lumina OH — Five Element Energy Rules

The Lumina OH system analyzes energy based on the traditional Five Elements.

Elements:

- Wood
- Fire
- Earth
- Metal
- Water

Each card contributes numeric values to the five elements.

Example:

Wood: 3  
Fire: 1  
Earth: 2  
Metal: 0  
Water: 4

---

# Energy Calculation

Step 1

User selects cards.

Step 2

Sum element values from all selected cards.

Example:

Wood = sum of all wood values  
Fire = sum of all fire values  
Earth = sum of all earth values  
Metal = sum of all metal values  
Water = sum of all water values  

---

# Energy Normalization

Convert total values into percentages.

Example:

Total energy = sum of all elements

Element percentage =

element_value / total_energy

---

# Energy Interpretation

Balance Range

Each element ideally falls between:

15% – 25%

This indicates a balanced profile.

---

# Dominant Energy

If an element exceeds:

30%

It is considered **dominant energy**.

Dominant energy indicates strong personality traits.

---

# Weak Energy

If an element falls below:

10%

It is considered **weak energy**.

Weak energy suggests areas requiring balance.

---
