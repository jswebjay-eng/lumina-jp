# Lumina OH — UI Design System (Award-Winning Aesthetic)

Lumina OH is designed as a **Digital Sanctuary**. The interface prioritizes emotional resonance, spiritual clarity, and the Japanese concept of **"Ma" (間)**.

---

# 1. Design Philosophy
- **Ma (間):** Intentional white space to allow the user's mind to rest.
- **Komorebi (木漏れ日):** Soft, layered lighting and gradients mimicking filtered sunlight.
- **Yugen (幽玄):** Depth and mystery through subtle blurs and glass-morphism.
- **Quietude:** Every interaction is designed to slow the user down.

---

# 2. Color System: "Natural Resonance"
- **Background:** `#FBF9F6` (Soft Washi)
- **Ink:** `#2A2A2A` (Charcoal)
- **Accents (The Five Elements):**
    - **Wood:** `#8BA889`
    - **Fire:** `#D98B73`
    - **Earth:** `#C4B08B`
    - **Metal:** `#B8BFC6`
    - **Water:** `#6B7B8C`

---

# 3. Typography: "Serenity & Clarity"
- **Headlines:** `Noto Serif JP` (Light/Medium). Tracking: `0.1em`.
- **Body:** `Noto Sans JP` (Light). Line-height: `1.8`.
- **Philosophy:** Evoke the elegance of vertical calligraphy while maintaining modern legibility.

---

# 4. Layout Grid: "Fluid Balance"
- **Grid:** 12-column system.
- **Max Width:** `1140px`.
- **Vertical Rhythm:** Large section gaps (`80px` - `120px`) to emphasize "Ma."

---

# 5. Component Library
- **Card UI:** 3:4 aspect ratio, `rounded-3xl`, "Floating Glass" effect.
- **Buttons:** Pill-shaped, minimal borders, soft background blurs.
- **Energy Report:** Organic radial charts with blurred "Energy Rings."

---

# 6. Motion Design: "Breath & Flow"
- **Duration:** `800ms` – `1200ms`.
- **Easing:** `cubic-bezier(0.22, 1, 0.36, 1)`.
- **Entrance:** Staggered fade-ins with a subtle upward drift (`20px`).
